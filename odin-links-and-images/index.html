Image Formats

JPG images can render many colours and gradients however are unable to generate transparent pixels

GIF are the simplest way to present short animations. They are limited for color, so do not use them for photos. Can have transparency but can't have sem-opaque colors.

PNG are great for anything that is not animated or photos. They can handle opacity and color palette. They are great for icons, logos, diagrams, etc.

SVG is vector based image. It can easily scale up and down in dimensions without loss of quality. They have the same use cases as PNG and are great for responsive designs. One caviat is if the image has a lot of text, it impacts the file size a lot.
